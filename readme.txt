din() 1.0
---------

The function

  din()
  
supplies an entry routine for a number in the terminal.

The given number will be stored in a variable by using like this:

  double a;
  
  a = din();
  
The function works exactly like the function  in()  however it uses

  double
  
as data type which is shown by using the "d" in the name of the function ("d" for "double")
